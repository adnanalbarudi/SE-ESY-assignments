# Circular Buffer - Assignment

## فكرة المشروع
بشكل عام المشروع هو تطبيق لفكرة الـ Circular Buffer بلغة C.  
الفكرة ببساطة إنو عندي مصفوفة بس مو خطية، يعني لما توصل للنهاية ما بتوقف، بترجع تكمل من البداية.

فينا نعتبرها طابور
أول عنصر بيدخل هو أول واحد بيطلع (FIFO)

استخدمت:
- مصفوفة من نوع char لحتى خزّن الحروف
- 3 متغيرات:
  - head  مكان القراءة
  - tail → مكان الكتابة
  - count → عدد العناصر جوّا البوفر


### الإدخال (write)
بضيف الحرف عند tail  
### الحذف (read)
بقرأ الحرف من head  
وبعدين بحرك

## ليش استخدمت count؟
بصراحة لحتى ما أتخربط بين حالتين:
- إذا البوفر فاضي
- إذا البوفر مليان  

لأنو أحياناً head و tail بيكونوا نفس الشي بالحالتين، فـ count حلّت المشكلة.
